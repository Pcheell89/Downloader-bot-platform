# 🤖 Multi-Platform Video Downloader Bot

Telegram-бот для скачивания видео из **TikTok**, **Instagram Reels** и **YouTube Shorts** без водяных знаков в наилучшем качестве.

## ✨ Возможности

- 🎵 **TikTok**
- 📸 **Instagram Reels**
- ▶️ **YouTube Shorts**
- 📁 **Большие файлы**
- 🕹️ **Удобное управление**
- 🚀 **Мгновенная работа**

## 🛠️ Технологии

- [Python](https://www.python.org/) 3.10+
- [Aiogram](https://docs.aiogram.dev/) 3.x — асинхронный фреймворк для Telegram Bot API
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) — загрузка Instagram Reels и YouTube Shorts
- [aiohttp](https://docs.aiohttp.org/) — HTTP-запросы к API TikTok
- [tikwm.com](https://www.tikwm.com/) — внешний сервис для получения прямых ссылок TikTok без водяных знаков
